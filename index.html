DevDepot

Dev-Depot was created with newcomers to writing code in mind. Using HTML, CSS, Javascript, and Express, Dev-Depot brings together templates for the various files needed to create a basic backend application. This was done so it would be easier for new coders to have everything set up correctly so the functional code can be focused on.

Wannabeer

Wannabeer is an easy way for anyone wanting a cold, locally produced beverage, to find somewhere to do so. It allows the user, upon letting the browser know his or her location, to find a brewery close to their location. Additionally, details about the selected brewery are displayed to allow the user to make a more informed choice.
